## 🚀 Key Features

- 🎵 Smooth music playback
- ⏯️ Play / Pause control
- ⏭️ Next / Previous track support
- 🔁 Loop (repeat current song)
- 📊 Interactive progress bar (seek control)
- 📜 Dynamic song list rendering
- ⚡ Lazy loading (batch loading for performance)
- 📱 Mini player interface
- 🎨 Clean and modern UI design

---

## 🛠️ Tech Stack

Frontend:
- HTML5
- CSS3
- JavaScript (Vanilla JS)

Hosting:
- Netlify

---

## ⚙️ How It Works

- Songs are managed using a JavaScript array
- Initial batch of songs loads for faster performance
- Additional songs load dynamically on scroll
- Audio element handles playback functionality
- UI updates are controlled via JavaScript events

## 🎯 Project Objective
